# Tech Stack

- **Backend:** PHP 8.4, Laravel 12
- **Frontend:** React 19, Inertia.js v2, Tailwind CSS v4
- **Testing:** Pest v4
- **Auth:** Laravel Fortify (headless)
- **Code Style:** Laravel Pint
- **Routing:** Laravel Wayfinder
- **Bundler:** Vite + @laravel/vite-plugin-wayfinder
- **Server:** Laravel Herd (local)
